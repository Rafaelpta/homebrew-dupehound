# homebrew-dupehound

Homebrew tap for [dupehound](https://github.com/Rafaelpta/dupehound), a fast, offline duplicate-code detector.

```sh
brew install rafaelpta/dupehound/dupehound
```

Then:

```sh
dupehound scan .
```

The formula tracks the release binaries published on the main repo. See [Rafaelpta/dupehound](https://github.com/Rafaelpta/dupehound) for what it does.
